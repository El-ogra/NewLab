# 🔹 الوظيفة الأولى: إضافة/تعديل بيانات مريض جديد

## 1. الوصول إلى النافذة
- **المسار**: النافذة الرئيسية → Toolbar العلوي → زر "المرضى" → زر "إضافة وتعديل بيانات المرضى"
- **الاختصار العالمي**: `F2` من أي نافذة رئيسية

---

## 2. تخطيط النافذة والأقسام

### القسم 1: أزرار الأوامر العلوية (المنطقة 1)
| الزر | الوظيفة |
|------|---------|
| **إضافة مريض** | تفريغ الحقول لإدخال مريض جديد |
| **تعديل** | تفعيل تعديل بيانات المريض المعروض حالياً |
| **حذف** | حذف المريض المحدد (يتطلب صلاحيات إدارية) |
| **إلغاء** | يظهر بدلاً من "إضافة/تعديل" عند بدء العملية للتراجع عنها |

> **ملاحظة**: عند الضغط على "إضافة" أو "تعديل"، يظهر زر "إلغاء" بدلاً منهما للتراجع.

---

### القسم 2: البيانات الأساسية للمريض (المنطقة 2)
| الحقل | الوصف | ملاحظات |
|-------|-------|---------|
| **اسم المريض** | نص حر | يُفضل إلغاء الهمزة، استبدال التاء المربوطة بهاء، استخدام الياء لا الألف المقصورة |
| **الجنس (Sex)** | Male / Female / Unknown | Radio buttons. اللقب يتغير تلقائياً حسب الجنس |
| **اللقب (Title)** | نص | يُعرض تلقائياً حسب الجنس، ويمكن تعديله يدوياً |
| **السن (Age)** | رقم عشري | يُكتب قبل اللقب حتى يعرض النظام المعدل الطبيعي المناسب |
| **وحدة السن** | Days / Months / Years | إذا < شهر: Days (1-29)، إذا < سنة: Months (1-11)، إذا ≥ سنة: Years (مع كسر، مثال 2.5 = سنتين ونصف) |

> **ملاحظة إملائية**: يجب توحيد طريقة كتابة الأسماء لسهولة البحث (أحمد=احمد، إبراهيم=ابراهيم، مروة=مروه، علي وليس على).

---

### القسم 3: نظام الحساب (المنطقة 3)
| النظام | الوصف |
|--------|-------|
| **Individual** | مريض مستقل، يحاسب بأسعار المرضى الخاصة بالمعمل حتى لو تم تحديد جهة إحالة له |
| **Lab to Lab** | إن وجدت قائمة أسعار للجهة → يحاسب بأسعارها. إن لم توجد → يحاسب بسعر "معمل لمعمل" الافتراضي |
| **Free** | مجاني تماماً، الإجمالي = صفر |

---

### القسم 4: مريض مهم (المنطقة 4)
- **VIP checkbox**: في حالة وجود مرضى مهمين ونتائجهم تشكل اهتماماً لإدارة المعمل
- يُفضل اختياره لسهولة الوصول لمثل هذه النتائج لاحقاً

---

### القسم 5: جهة الإحالة (المنطقة 5)
- **حقل نصي**: اسم جهة الإحالة
- **Autocomplete**: عند بداية كتابة الاسم تُفعل خاصية التكميل التلقائي
- **حفظ تلقائي**: عند إضافة جهة جديدة يتم حفظها تلقائياً في النظام
- **جهة افتراضية**: في حالة عدم إدخال اسم جهة إحالة، يسجل النظام "المعمل" تلقائياً
- **خصم الجهة**: في حالة اختيار اسم جهة لها نسبة خصم، يتم إضافة نسبة الخصم تلقائياً في الحسابات (من نافذة الجهات الخارجية والمعامل)

---

### القسم 6: إخفاء الجهة على النتيجة (المنطقة 6)
- **Checkbox**: "نزيل اختيار الطباعة"
- في حالة التعامل مع جهات Lab to Lab وعدم رغبة ظهور اسم الجهة على النتيجة

---

### القسم 7: معلومات إضافية طبية (المنطقة 7)
| الحقل | الوصف |
|-------|-------|
| **ساعات الصيام** | عدد ساعات الصيام |
| **أدوية سيولة** | هل يتناول أدوية سيولة؟ |
| **علاج كبد** | هل يوجد علاج كبد؟ |
| **علاج فيروسات** | هل يوجد علاج فيروسات؟ |
| **مضاد حيوي** | هل يوجد مضاد حيوي؟ |
| **حمل** | هل يوجد حمل للسيدات؟ |
| **تدخين** | هل المريض مدخن؟ |

---

### القسم 8: نوع العينة (المنطقة 8)
- **الغرض**: في حالة إرسال العينة للمعمل وسحبها خارج المعمل
- **الخيارات**: Blood, Urine, Stool, Semen, CSF, Other
- **التأثير**: يظهر تعليق في التقرير يوضح أن العينة مأخوذة خارج المعمل

---

### القسم 9: ملاحظة (المنطقة 9)
- **حقل نصي كبير**: ملاحظة عن المريض أو عينات متأخرة
- **الغرض**: تعليق من الاستقبال إلى المعمل (والعكس) ليطلع عليه الطبيب أثناء إجراء التحاليل

---

### القسم 10: قائمة التحاليل المتاحة (المنطقة 10)
- **Dropdown**: اختيار نوع القائمة المعروضة:
  - التحاليل الروتينية
  - كل التحاليل المخزنة
  - مجموعات التحاليل (تحاليل ما قبل الزواج، قبل علاج فيروس سي، مجموعات التشكاب...)
  - المجموعات المخصصة الخاصة بالمعمل
- **مربع بحث**: بحث بالاسم أو رقم التحليل
- **الإضافة**: بالضغط مرتين على التحليل، أو زر "إضافة التحليل"، أو بالضغط على `F1` لعرض بيانات التحليل أولاً

---

### القسم 11: قائمة التحاليل المختارة (المنطقة 12)
- **العرض**: قائمة بأسماء التحاليل المختارة للمريض
- **الحذف**: بالضغط مرتين على التحليل
- **حذف الكل**: زر "Delete" أو "All"
- **الحساب**: يُحدَّث تلقائياً حسب نظام الحساب المختار

---

### القسم 12: حساب المريض (المنطقة 13)
| العنصر | الوصف |
|--------|-------|
| **الخصم** | برقم صحيح أو نسبة % |
| **المبلغ المدفوع** | المبلغ الذي دفعه المريض |
| **زر "خالص"** | لتصفية حساب المريض |
| **زر "موافق"** | للتأكيد النهائي |
| **Enter مرتين** | في الكيبورد للتأكيد والدفع |

---

### القسم 13: Lab ID (المنطقة 14)
- **زر "Lab ID"**: لتسجيل كود المعمل الدائم للمريض (بطاقة باركود)
- **الاستخدام**: عند الزيارة القادمة، يُقرأ الباركود من خانة "Lab ID" ليستدعي بيانات المريض
- **في حالة فقدان البطاقة**: يُفضل عمل بحث عن المريض من نافذة البحث وطباعة الباركود مرة أخرى (دون إعطاء رقم جديد)

---

### القسم 14: مرضى اليوم (المنطقة 15)
- **زر "مرضى اليوم"**: يفتح قائمة منسدلة بمرضى اليوم
- **الاستخدام**: تعديل بياناتهم أو تحصيل مبالغ مالية
- **ملاحظة**: يمكن تسديد المبالغ أيضاً من نافذة "تسليم النتائج"

---

### القسم 15: الإيصال
- **الطباعة**: طباعة إيصال بالإجمالي والخصم والمطلوب سداده والباقي
- **التفاصيل**: يمكن اختيار ظهور تفصيل التحاليل وأسعارها في الوصل
- **التلقائي**: يمكن ضبط الإيصال ليُطبع تلقائياً بعد حفظ المريض (من نافذة إعدادات النظام → إعدادات الإيصال)

---

## 3. اختصارات لوحة المفاتيح في النافذة

| المفتاح | الوظيفة |
|---------|---------|
| `F1` | عند وضع المؤشر في قائمة التحاليل → تظهر بيانات التحليل المحدد |
| `F2` | إضافة مريض جديد |
| `F3` | الانتقال لنافذة البحث |
| `F4` | الانتقال لنافذة إدخال النتائج |
| `F5` | Refresh (تحديث قائمة التحاليل والتهيئة لبحث جديد) |
| `F6` | الانتقال لنافذة تسليم النتائج |
| `F7` | الانتقال لنافذة العينات المرسلة للخارج |
| `F8` | تعديل بيانات المريض المحدد |
| `F9` | حفظ المريض الجديد أو التعديل |
| `F10` | حذف المريض المحدد |
| `F11` | الانتقال لنافذة طباعة الباركود للمريض المحدد |
| `F12` | طباعة إيصال للمريض المحدد |
| `Enter` | الانتقال بين مربعات إدخال البيانات، تحصيل ثمن التحاليل، إضافة التحليل/المجموعة المحددة |
| `Up/Down` | الانتقال بين مربعات إدخال البيانات، التنقل في قائمة التحاليل |
| `Ctrl` | الانتقال بين مربع البحث عن تحليل/مجموعة وقائمة التحاليل |
| `Esc` | التراجع عن إضافة مريض جديد أو تعديل مريض سابق، العودة للقائمة الرئيسية |

# 🔹 الوظيفة السابعة: إضافة/تعديل بيانات تحليل

## 1. الوصول إلى النافذة
- **المسار**: النافذة الرئيسية → Toolbar العلوي → زر "بيانات النظام" → زر "بيانات التحاليل"
- **لا يوجد اختصار لوحة مفاتيح مذكور**

---

## 2. تخطيط النافذة والأقسام

### القسم 1: قائمة التحاليل (المنطقة 1)
- **العرض**: جدول يحتوي على كل التحاليل المسجلة في النظام
- **الأعمدة**: ID, Arrange, Group Name, Test Name, Pat. P, Lab P, Out Lab Name, Out P, Barcode, Routine
- **الاختيار**: بالضغط على التحليل بالفأرة لعرض بياناته الكاملة في القسم 3

---

### القسم 2: البحث (المنطقة 2)
- **3 خانات للبحث** (من اليسار إلى اليمين):
  1. كود التحليل
  2. اسم المجموعة
  3. اسم التحليل
- **النتيجة**: تعرض في قائمة التحاليل (القسم 1)

---

### القسم 3: بيانات التحليل (المنطقة 3)

#### حقول النص الأساسية (17 حقل):
| الحقل | الوصف |
|-------|-------|
| **Test name** | الاسم المتعارف عليه داخل المعمل (اسم البرنامج الداخلي) |
| **Report name** | اسم التحليل في التقرير. يُقسم لجزأين: الأول بخط كبير والثاني بخط صغير. مثال: SGPT/ALT (Alanine Aminotransferase). يمكن استخدام خانة واحدة فقط |
| **Bill name** | اسم التحليل في الوصل (الإيصال). يُقسم لجزأين. عند تشابه الجزء الأول يقوم النظام بتجميع التحاليل في عنصر واحد (مثل: تحاليل الكبد والكلى) |
| **History name** | اسم مختصر في تاريخ المريض. يُفضل أن يكون مختصراً لصغر المسافة |
| **Arabic name** | الاسم بالعربي |
| **Group name** | اسم المجموعة المنتمي إليها التحليل. تفيد في الإحصائيات وعند إضافة مجموعة تحاليل كاملة للمريض |
| **Branch** | فرع المعمل المسؤول عن هذا التحليل |
| **Log group** | اسم مجموعة العمل (الشغل). مثال: في حالة وجود جهازين (كيمياء كبير وصغير)، تحليل السكري على الجهاز الأول. عند طلب "ورقة عمل" لعينات اليوم بنظام مجموعات العمل، يطبع التحاليل حسب الجهاز |
| **Collection** | ملاحظات عن شروط سحب العينة |
| **Test time** | الوقت المستغرق لإنهاء التحليل (بالأيام). صفر = نفس اليوم. يعتمد عليه وقت استلام النتيجة في الوصل |
| **Arrange number** | رقم ترتيب التحليل في قوائم التحليل. في حالة إدخال أكثر من تحليل بنفس الرقم يكون الترتيب بالاسم |
| **Patient price** | سعر التحليل للمريض العادي (المستقل) |
| **Lab to Lab price** | سعر التحليل للمعامل والجهات الأخرى (في حالة عدم وجود قائمة أسعار مخصصة للجهة) |

#### خصائص التحليل (Checkboxes):
| الخاصية | الوصف |
|---------|-------|
| **Routine test** | جعل التحليل يظهر في قائمة التحاليل الروتينية (الأكثر طلباً) في نافذة بيانات المرضى |
| **See report** | تحليل له أكثر من نتيجة (مثل البول/البراز/صورة الدم). لا يمكن عرض النتيجة عند بيانات المريض |
| **Print with other** | في حالة التحاليل الكبيرة (مثل البول/البراز) يمكن طباعتها في تقرير مجمع مع تحاليل أخرى |
| **Add with group** | في حالة اختيار مجموعة تحاليل للمريض، يُضاف هذا التحليل تلقائياً مع المجموعة. مثال: الجاماجلوبين يُطلب كثيراً مع إنزيمات الكبد، فعند إضافة المجموعة يُضاف تلقائياً |
| **Main test** | تحليل رئيسي له عناصر فرعية (مثل غازات الدم/هيموجلوبين/فيدال). في العناصر الأخرى تُلغى هذه العلامة |

---

### القسم 4: Barcode ونوع العينة (المنطقة 4)
- **Name**: اسم التحليل على الملصق
- **Tube 1, 2, 3**: تحديد نوع الحاوية/المادة المسحوبة لطباعة ملصق باركود خاص
- **مثال**: 
  - البول = ملصق واحد (مادة بول)
  - Creatinine Clearance = مصل + بول 24 ساعة (ملصقان)

---

### القسم 5: إرسال للخارج (المنطقة 5)
- **Checkbox**: في حالة إرسال التحليل لخارج المعمل
- **Lab Name**: اسم الجهة/المعمل المرسل إليه
- **Cost price**: سعر التكلفة
- **التأثير**: عند إدخال مريض لهذا التحليل، يظهر في نافذة "العينات المرسلة للخارج". عند إرساله يُدخل في حسابات المعمل

---

### القسم 6: أسئلة المريض (المنطقة 6)
- **Patient Question**: سؤال/تنبيه يظهر للمريض عند اختيار هذا التحليل
- **مثال**: عند اختيار تحاليل الدهون → "هل أنت صائم؟ يجب أن تكون صائماً على الأقل 12 ساعة"

---

### القسم 7: لوحة الحروف اللاتينية (المنطقة 7)
- **الرموز المتاحة**: α, β, γ, μ, ±, ≤, ≥, °
- **الاستخدام**: أثناء إدخال الأسماء، الضغط على الحرف المراد لإدراجه في خانة النص
- **يمكن أيضاً**: نسخ ولصق في خانات النصوص الأخرى

---

### القسم 8: أزرار الأوامر (المنطقة 8)
| الزر | الوظيفة |
|------|---------|
| **إضافة تحليل** | إضافة تحليل جديد إلى النظام |
| **تعديل** | تعديل بيانات التحليل المحدد |
| **حفظ** | حفظ التحليل الجديد أو المعدل |
| **تراجع** | التراجع عن العملية الجارية |
| **حذف** | حذف التحليل المحدد |
| **المعدل الطبيعي** | الانتقال لنافذة المعدلات الطبيعية للتحليل المحدد |
| **قائمة التحاليل** | الرجوع لقائمة التحاليل المسجلة |

---

## 3. ملاحظات خاصة

### تعديل عناصر تحليل رئيسي
- لتعديل بيانات أو المعدلات الطبيعية لمجموعة تحاليل فرعية (مثل عناصر صورة الدم أو الفيدال):
  1. أدخل أول حرفين من اسم التحليل في خانة البحث الثالثة (الخاصة بالبحث باسم التحليل)
  2. يقوم النظام بعرض جميع العناصر لهذا التحليل
  3. اختر العنصر بالفأرة

### لا توجد اختصارات لوحة مفاتيح مذكورة لهذه النافذة.
